Overview

simple_calculus is a production-ready numerical calculus library for Eiffel providing derivatives, integrals, and partial derivatives with verified accuracy.

It offers three-tier API design (simple/advanced/expert), adaptive algorithms with automatic step size selection, and convergence analysis through three-method comparison.

Part of the Simple Eiffel ecosystem.

Features

  • Tier 1 API ‚Äî Simple interface for derivatives, integrals, and partial derivatives
  • Tier 2 API ‚Äî Advanced control with method selection and error bounds
  • Tier 3 API ‚Äî Expert access to computation engines for research use
  • Adaptive Algorithms ‚Äî Automatic step size and tolerance management
  • Convergence Analysis ‚Äî Three-method comparison for derivative accuracy
  • SCOOP Compatible ‚Äî Immutable results enable concurrent use

Quick Example

Compute the derivative of f(x) = x¬≤ at x = 1:

local
    l_calc: SIMPLE_CALCULUS
    l_f: FUNCTION [REAL_64, REAL_64]
do
    create l_calc
    l_f := agent (x: REAL_64): REAL_64 do Result := x * x end
    Result := l_calc.derivative (l_f, 1.0)  -- Returns ~2.0
end

See Quick API for more examples.

Installation

Add simple_calculus to your ECF configuration file:

<library name="simple_calculus" location="$SIMPLE_EIFFEL/simple_calculus/simple_calculus.ecf"/>

Then use in your code:

create l_calc
